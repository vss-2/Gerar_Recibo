### Requerimentos:
* Python3  (3.6.9)
* BottlePy (1.3-dev)
* PIL - Python Image Library (7.2.0)

* Documentação: [BottlePy](https://bottlepy.org)
  
### Instalação
* ```pip install bottle```
* ```pip3 install Pillow```
* Download do Bottle no [PyPi](https://pypi.python.org/pypi/bottle)

### Execução 
* Execute: ```python3 NFE_filecreator.py```
* Abra: ```http://localhost:8080/inicio```
